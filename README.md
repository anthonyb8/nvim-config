# Nvim-config

# Install NeoVim
```
brew install nvim 
```

# Clone Config

```
cd ~/.config

git clone https://github.com/anthonyb8/nvim-config.git ~/.config/nvim
```

# Install Tmux
```
brew install tmux
```

# Tmux config
```
mv ~/.config/nvim/tmux.conf ~/.tmux.conf
```

# Install Tmux plugins
```
Ctrl + a | Ctrl + i
```
