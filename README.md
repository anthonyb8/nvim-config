## Install NeoVim
#### MacOS
```
brew install neovim
```

#### Linux
```
sudo apt update
sudo apt install neovim
```

## Clone Config
```
git clone https://github.com/anthonyb8/nvim-config.git ~/.config/nvim
```

## Install Tmux
#### MacOS
```
brew install tmux
```
#### Linux
```
sudo apt install tmux
```

## Tmux config
```
mv ~/.config/nvim/tmux.conf ~/.tmux.conf
```

## Install Tmux plugins
```
Ctrl + a | Ctrl + i
```
